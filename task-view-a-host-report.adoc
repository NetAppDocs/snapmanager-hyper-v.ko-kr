= 호스트 보고서를 봅니다
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ./media/


[role="lead"]
SnapManager for Hyper-V에서 관리되는 호스트에 대한 보고서를 볼 수 있습니다

.단계
. 탐색 창에서 보고서 [호스트] 메뉴를 클릭합니다.
. 보려는 보고서가 포함된 호스트를 선택합니다.
. 보고서 창에서 보려는 보고서를 선택하고 * 보고서 보기 * 를 클릭합니다.
+
구성 보고서가 별도의 창에 나타납니다.
