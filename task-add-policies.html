    • 다음 VM에 대한 백업이 실패하였습니다. 온라인 백업 또는 백업 대상 VM을 찾을 수 없기 때문입니다
    • IVssWriterCallback 인터페이스에 대한 예기치 않은 쿼리 오류. hr=0x80070005, 액세스가 거부되었습니다.
    • 백업 보고서는 보고서 이름에 관리 콘솔 시간대 정보를 사용합니다
    • IPv6 전용 환경에서는 백업 및 복원 알림이 전송되지 않습니다
    • 장애 조치 클러스터링 이벤트 ID 5121
    • 복구 작업이 진행 중인 동안 생성된 가상 머신 백업이 잘못되었을 수 있습니다
    • 가상 시스템 관리
    • IPv6 전용 호스트에서는 연결 시간이 길어집니다
    • 볼륨 섀도 복사본 서비스 오류: 내부 불일치가 감지되었습니다
    • 웹 서비스 클라이언트 채널이 시스템 smhv51_81clus의 ConfigurationManagementService 인스턴스에 연결할 수 없습니다
    • MSI 사용자 정의 속성은 자동 설치에 사용됩니다
    • SnapManager for Hyper-V는 호스트 또는 스토리지 시스템에서 라이센스가 부여되지 않습니다
    • 페일오버 후 백업을 삭제합니다
    • 백업 실패 후 스토리지 성능이 저하됩니다
    • SnapInfo 스냅샷 복사본이 삭제되었습니다
    • 바이러스 백신 솔루션으로 인한 높은 메모리 소비
    • 각 백업에 대해 2개의 스냅샷 복사본을 생성할 때의 공간 소비
    • 백업하는 동안 SnapDrive SDDiscoveryFileSystemListInfo 응답이 null입니다
    • 오류: VSS 요청자 - 백업 구성 요소에 실패했습니다
    • VSS 요청자 - 백업 구성 요소 실패. 예상 디스크가 시스템에 도착하지 않았습니다
    • VSS 요청자 - 부분 작성기 오류로 백업 구성 요소가 실패했습니다
    • VSS는 NAS 백업 중에 Microsoft iSCSI Target VSS Hardware Provider에 대한 오류를 반환합니다
    • VSS 요청자 - 백업 구성 요소 실패. 스냅샷 세트 유지를 호출하지 못했습니다.
    • SnapManager for Hyper-V에서 MBR LUN이 지원되지 않습니다
    • Hyper-V Manager에서 가상 컴퓨터를 제거한 후 백업이 실패합니다
    • 일부 유형의 백업 실패는 부분 백업 실패를 초래하지 않습니다
    • 스토리지 시스템 볼륨 이름 변경 후 복구 실패
    • 페일백 후 백업에서 복원합니다
    • 데이터 세트를 새 노드로 업데이트하는 동안 웹 서비스 클라이언트 채널을 연결할 수 없습니다
    • 데이터 세트는 Windows 페일오버 클러스터의 새 노드로 자동 복제되지 않습니다
    • 오류 1935. 어셈블리 부품을 설치하는 동안 오류가 발생했습니다
    • 동일한 스토리지 시스템에서 15개 이상의 CSV를 포함하는 백업 작업이 실패할 수 있습니다
    • 지정된 VM이 없거나 온라인으로 백업할 수 없습니다
