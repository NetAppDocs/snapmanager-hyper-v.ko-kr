= SMB를 통한 Hyper-V에 대해 SVM 또는 CIFS 서버 구성
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
여러 애플리케이션을 위해 단일 SVM(스토리지 가상 머신) 또는 CIFS 서버를 구성하면 리소스 공유 문제가 발생할 수 있으며, 이로 인해 Hyper-V 환경이 영향을 받을 수 있습니다. 요구 사항에 따라 SMB를 통한 Hyper-V용 전용 SVM 또는 CIFS 서버를 구성해야 합니다.
